EntropySource abstract class

An asynchronous source of pure random data (entropy). Data returned by an EntropySource should be supposed to be unpredictable and of more quality than data obtained from a SecureRandom. Usually an EntropySource should be seen like a random generation device while a SecureRandom should be seen like a cryptographic PRNG.
