cipher

A Dart library for encryption and decryption

Download this project as a .zip file Download this project as a tar.gz file

Cipher is a Dart library for encryption and decryption. As of today, most of the classes are ports of Bouncy Castle from Java to Dart.

To make sure nothing fails, tests and benchmarks for every algorithm are provided. The expected results are taken from the Bouncy Castle Java version and also from standards, and matched against the results got from cipher.

To use the package follow the instructions in the Installation section of the package in Pub.

To get directions on how to use the library consult the project's wiki in GitHub.

You can also browse cipher API's documentation online (there's also an alternative documentation exposing the API as well as the implementations).

To ask questions about the project or get info about releases visit or subscribe to the project's mailing list.

To see the list of implemented algorithms and technical specifications see the README.md file in GitHub.
